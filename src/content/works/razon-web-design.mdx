Razón Pública es un portal de noticias en línea dedicado a ofrecer una cobertura completa de noticias de Colombia. Este proyecto muestra una combinación perfecta de diseño y funcionalidad, desarrollado en WordPress y meticulosamente creado en Figma para proporcionar una experiencia de usuario excepcional.

### Características Clave

-   **Portal de Noticias:** Razón Pública ofrece noticias actualizadas y artículos informativos que cubren una amplia gama de temas relevantes para Colombia. El diseño prioriza la legibilidad y la facilidad de navegación, asegurando que los usuarios puedan acceder fácilmente a las últimas noticias.
-   **Diseño UI/UX Personalizado:** Toda la interfaz de usuario y la experiencia de usuario fueron diseñadas a medida en Figma, centrándose en una estética moderna y limpia que mejora el compromiso del usuario. El proceso de diseño involucró una extensa investigación de usuarios y pruebas iterativas para lograr un diseño que sea tanto visualmente atractivo como altamente funcional.
-   **Integración con WooCommerce:** Una tienda WooCommerce completamente integrada soporta la venta en línea de varios productos. Esto incluye una experiencia de compra amigable con listados de productos intuitivos, páginas de productos detalladas y un proceso de pago fluido.
-   **Tipos de Publicaciones Personalizadas para Obras de Arte:** Además de las publicaciones de noticias estándar, se crearon tipos de publicaciones personalizadas específicamente para mostrar piezas de arte. Esta característica permite presentaciones detalladas de obras de arte, incluyendo imágenes de alta calidad, información del artista y opciones de compra.
-   **Diseño Responsivo:** El sitio fue diseñado para ser totalmente responsivo, asegurando un rendimiento y una estética óptimos en todos los dispositivos, desde computadoras de escritorio hasta teléfonos móviles. Esto garantiza que los lectores puedan acceder al contenido de Razón Pública sin problemas, sin importar dónde se encuentren.

![Razon Pública Pages](@assets/images/razon-internas.jpg)

El proceso de desarrollo aprovechó las capacidades robustas de WordPress para asegurar una plataforma escalable y mantenible. Se utilizaron tipos de publicaciones personalizadas, plantillas a medida y varios plugins para mejorar la funcionalidad y la experiencia del usuario en el sitio.
